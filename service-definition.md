# 服务定义



## 1. 主服务
- 名称：公共治安综合平台
- 代号：picasso
- 简介：基础功能（用户、权限体系）和核心功能（场所备案、从业人员备案），从业人员子平台 Web API 和 行政管理子平台 Web API。
- 文档仓库：picasso-design
- 代码仓库：picasso-source
- 部署仓库：picasso-deploy
